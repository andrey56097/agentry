## Evaluation criteria
- Correctness: Does it do what the task asks?

- Completeness: Are all necessary parts implemented?

## Output format

```json

```

## Score interpretation

- 41-70: Functional but needs improvements
- 71-90: Good, minor issues
- 91-100: Excellent, ready to ship
